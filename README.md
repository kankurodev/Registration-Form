## Description
This application is a registration form that utilized advanced validation features to verify the input fields contain valid input.

## Live Preview
You can view the live preview of this project [here](https://tobyhorn.github.io/Registration-Form/)

## How To
Fill out the form and click submit. If the input fields contain any errors you will be notified, if not then a success display will appear.
